## Installation

#### Backend

Need Mongodb installed first

```bash

$ cd /api

# install dependencies
$ npm install

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

See the app running at `http://localhost:3000/`

#Docker 
##Docker dev

$ sudo docker-compose up

#Docker 
##Docker prod

$ sudo docker-compose up 

```

#### Frontend

```bash

$ cd /app

# install dependencies
$ ng serve

# production mode / build
$ ng build

See the app running at `http://localhost:4200/`

```
