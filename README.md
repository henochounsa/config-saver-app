## Installation

#### Backend

Need Mongodb installed first

```bash

$ cd /testapp-api

# install dependencies
$ npm install

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

See the app running at `http://localhost:3000/`

```

#### Frontend

```bash

$ cd /testapp

# install dependencies
$ ng serve

# production mode / build
$ ng build

See the app running at `http://localhost:4200/`

```
